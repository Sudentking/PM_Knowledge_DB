# Agent.md - 项目进度记录

## 项目概述

**项目名称**：产品经理知识库 (Product Manager Knowledge DB)

**项目目标**：创建一个自动化的产品经理知识管理系统，从人人都是产品经理网站（woshipm.com）爬取文章，通过AI生成详细摘要，并同步到GitHub远程仓库。

**技术栈**：
- Python 3.9+
- requests + BeautifulSoup (爬虫)
- OpenAI GPT-4 (AI总结)
- Git + GitHub (版本控制)
- YAML (配置管理)

## 技术决策记录

### 2026-05-26
- **决策**：选择requests + BeautifulSoup作为爬虫框架
- **理由**：轻量级，适合静态页面，学习成本低

- **决策**：选择OpenAI GPT-4作为AI总结服务
- **理由**：生态成熟，文档完善，中文支持好

- **决策**：选择Markdown作为知识库存储格式
- **理由**：可读性强，便于版本控制，任何平台都能读取

## 问题和解决方案

### 问题1：网站反爬机制
**描述**：目标网站可能有反爬机制，导致爬取失败
**解决方案**：
- 添加User-Agent伪装
- 设置合理的爬取间隔
- 支持代理配置

### 问题2：AI API限流
**描述**：OpenAI API可能有调用频率限制
**解决方案**：
- 添加重试机制
- 实现批量处理
- 支持多个API Key轮换
