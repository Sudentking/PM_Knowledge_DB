# 添加新爬取分类操作指南

## 概述

本指南说明如何添加新的 woshipm.com 分类到知识库系统。添加新分类不会影响已有分类。

## 操作步骤

### 1. 修改爬虫配置

编辑 `config/config.yaml`，在 `crawler.categories` 列表中添加新的分类slug：

```yaml
crawler:
  categories:
    - "pd"           # 产品设计
    - "it"           # 信息技术
    - "ai"           # 人工智能
    - "share"        # 分享
    - "class"        # 课堂
    - "operate"      # 运营
    - "新分类slug"    # 新分类
```

### 2. 修改分类配置

编辑 `config/categories.yaml`，添加新分类的详细配置：

```yaml
  新分类slug:
    name: "分类中文名"
    description: "分类描述"
    url_slug: "新分类slug"
    keywords:
      - "关键词1"
      - "关键词2"
      - "关键词3"
```

### 3. 修改总结器映射

编辑 `scripts/summarizer.py`，在 `category_mapping` 字典中添加映射：

```python
category_mapping = {
    'pd': 'product-design',
    'user-research': 'user-research',
    'data-analysis': 'data-analysis',
    'project-management': 'project-management',
    'strategy': 'strategy',
    'industry-insights': 'industry-insights',
    'operate': 'operate',
    '新分类slug': '新分类目录名'  # 新增
}
```

### 4. 创建知识库目录

创建 `knowledge_base/新分类目录名/README.md`：

```markdown
# 分类中文名

本目录包含XXX相关的文章总结。

## 分类说明

- **子分类1**：说明
- **子分类2**：说明

## 文件组织

```
新分类目录名/
├── README.md          # 本文件
└── YYYY-MM-DD/        # 按日期归档
    └── article-title.md
```

## 更新日志

- 添加XX分类：YYYY-MM-DD
```

### 5. 测试新分类

运行爬虫测试：

```bash
python scripts/crawler.py --category 新分类slug --max-pages 1
```

## URL格式说明

woshipm.com 的分类URL格式为：
- 列表页：`https://www.woshipm.com/category/{slug}`
- 文章页：`https://www.woshipm.com/{slug}/{文章ID}.html`

## 常见分类slug参考

| slug | 分类名称 |
|------|----------|
| pd | 产品设计 |
| it | 信息技术 |
| ai | 人工智能 |
| share | 分享 |
| class | 课堂 |
| operate | 运营 |
| user-research | 用户研究 |
| data-analysis | 数据分析 |
| project-management | 项目管理 |
| strategy | 产品策略 |
| industry-insights | 行业洞察 |

## 注意事项

1. 分类slug必须与网站URL中的slug一致
2. 添加后无需修改工作流脚本，系统会自动遍历所有配置的分类
3. 知识库目录名建议使用英文小写加连字符格式
